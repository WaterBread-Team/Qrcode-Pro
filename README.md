# Qrcode-Pro

Aplicación en Python para generar códigos QR de forma rápida y sencilla.

---

## 🚀 Descargar y usar (fácil)

1. Ve a la carpeta `dist/`
2. Descarga `QrPro.exe`
3. Ejecuta el archivo (doble clic)

⚠️ No necesitas instalar Python

---

## 🧑‍💻 Usar desde código (opcional)

### 1. Clonar repositorio

```bash
git clone https://github.com/tuusuario/Qrcode-Pro.git
cd Qrcode-Pro
```

### 2. Instalar dependencias

```bash
pip install -r requirements.txt
```

### 3. Ejecutar

```bash
python QRPO.py
```

---

## 📦 Requisitos

* Python 3 (solo si ejecutas el código)

---

## 🧠 Funcionalidades

* Generación de códigos QR
* Compatible con texto y enlaces
